## ğŸ’¡ What Makes It Work

### Real-Time Communication
- WebSocket (Socket.IO) for instant updates
- < 50ms latency for sentiment updates
- < 100ms latency for MCQ broadcast

### Role-Based Access
- Instructor: `isAdmin: true` (dashboard access)
- Participant: `isAdmin: false` (voting access)
- Detected via URL: `?role=instructor` or `?role=participant`

### Persistent UI
- Sentiment buttons always visible for participants
- Real-time updates for all connected users
- Live participant list with sentiment status

### AI Integration
- Gemini API for MCQ generation
- Fallback to mock MCQs if API unavailable
- Natural language prompts supported
